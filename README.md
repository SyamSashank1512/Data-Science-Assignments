## How to Push One by One
To push a single pair to GitHub, you can use the following commands:
1. `git add Assginment_Codes/ProjectName.ipynb`
2. `git add Assginment_CSV/ProjectName.csv`
3. `git commit -m "Add ProjectName assignment"`
4. `git push origin main`
